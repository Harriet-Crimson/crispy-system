**User story: 1B**
Precondition: Pre-existing study profile
Input: click
Action: Select a module
Expected Result:
- Module page is displayed
- Corresponding assignments are shown, including deadlines
- Response time < 3 seconds


**User story: 1B**
Precondition: Pre-existing study profile
Input: click
Action: Select study dashboard
Expected Result:
- Study dashboard is displayed
- Categorised assignments based on deadlines and completion status
- Progress bars displayed for each deadline, showing approximate correct percentage
- Displays Gantt chart representation of student-entered data
- Response time < 4 seconds

**User story: 1C**
Precondition: Pre-existing task
Input: valid activity, at least time taken
Action: Submit activity to task
Expected Result:
- Progress is added
- Progress bar updated
- Response time < 1 second

**User story: 2B**
Precondition: pre-existing task
Input: new date
Action: change deadline
Expected result:
- The deadlines for all students assignment change to the user input date
- Response time < 1 minute

**User story: 2C**
Precondition: pre-existing module
Input: task details
Action: submit task
Expected result:
- All students can view the task
- Response time < 1 minute
